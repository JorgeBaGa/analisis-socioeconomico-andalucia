# Análisis Socioeconómico de Andalucía

Repositorio de datos y análisis socioeconómico de Andalucía a nivel municipal.

## Fuentes de datos
- **BADEA** (Banco de Datos Estadísticos de Andalucía) - IECA Junta de Andalucía

## Estructura
```
datos/
└── badea/          # 454 tablas estadísticas de todos los municipios de Andalucía
notebooks/          # Cuadernos de descarga y procesamiento
analisis/           # Análisis temáticos
```

## Temáticas incluidas
1. Territorio y medio ambiente
2. Demografía y población
3. Economía y mercado laboral
4. Servicios sociales y educación
5. Infraestructuras
6. Sector agrario

## Actualización
Datos descargados automáticamente via API REST de BADEA.
